นักเรียนสายวิทย์–คณิต | สนใจเทคโนโลยีสารสนเทศ (IT)

ประวัติส่วนตัวและการศึกษา

  • โรงเรียน: บุญวาทย์วิทยาลัย
  • แผนการเรียน: วิทยาศาสตร์–คณิตศาสตร์
  • ระดับชั้น: มัธยมศึกษาปีที่ 4
  • เกรดเฉลี่ย: 3.81
  • วิชาที่สนใจ: คอมพิวเตอร์ อังกฤษ วิทยาศาสตร์

แรงบันดาลใจด้าน IT

ข้าพเจ้ามีความสนใจด้านเทคโนโลยีสารสนเทศจากการเรียนวิชาคอมพิวเตอร์ และการใช้เทคโนโลยีในชีวิตประจำวัน ทำให้เห็นว่าไอทีมีบทบาทสำคัญ ในการแก้ไขปัญหาและพัฒนาคุณภาพชีวิต โดยเฉพาะการเขียนโปรแกรม และการคิดวิเคราะห์อย่างเป็นระบบ

ทักษะและความรู้

ภาษาโปรแกรม

  • Python (พื้นฐาน)
  • C / C++ (เบื้องต้น)
  • HTML / CSS
  • JavaScript (พื้นฐาน)
  • Scratch

เครื่องมือ

  • Visual Studio Code
  • Microsoft Word / Excel
  • Canva / Photoshop
  • Git / GitHub (พื้นฐาน)

ทักษะอื่น ๆ

  • การคิดวิเคราะห์
  • การแก้ปัญหาอย่างเป็นขั้นตอน
  • การเรียนรู้ด้วยตนเอง
  • การทำงานร่วมกับผู้อื่น

ผลงาน / โครงงาน

อยู่ระหว่างการพัฒนาและสะสมผลงานด้านเทคโนโลยีสารสนเทศ

เป้าหมายในอนาคต

มีเป้าหมายศึกษาต่อด้านเทคโนโลยีสารสนเทศ เพื่อพัฒนาทักษะการเขียนโปรแกรม และนำความรู้ไปใช้ในการทำงานจริง รวมถึงการพัฒนาสังคมด้วยเทคโนโลยี
